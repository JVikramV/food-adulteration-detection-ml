# 🧪 Food Adulteration Detection using Machine Learning

An end-to-end machine learning system to detect food adulteration in **chilli powder (image-based)** and **milk (spectral data-based)**. The system provides real-time predictions, interpretability via heatmaps, and an interactive web interface.

---

## 🚀 Features

### 🌶️ Chilli Powder Analysis
- Predicts **percentage of adulteration** from input images
- Generates **Grad-CAM heatmaps** to highlight adulterated regions
- Classifies samples as:
  - ✅ Pure
  - ⚠️ Adulterated

### 🥛 Milk Analysis
- Uses **spectral data (CSV input)**
- Detects:
  - Presence of adulteration
  - Type of adulterant
- Visualizes **spectral curves for analysis**

---

## 🧠 Tech Stack

- **Machine Learning:** PyTorch, Scikit-learn  
- **Data Processing:** NumPy, Pandas  
- **Visualization:** Matplotlib, OpenCV  
- **Frontend/UI:** Gradio  
- **Backend:** Python
